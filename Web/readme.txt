================================
Building / Setting Up
================================

Server:
go to /Klampt/Web/Server directory
make

Client:
make sure you have Bower installed via NPM
go to /Klampt/Web/Client directory, and run:
bower install

================================
Running
================================

Start up the server
goto /Klampt directory
./WebServer

Start up a client
goto /Klampt/Web/Client
click on index.html
make sure server URL is correct
click connect
click submit code
click run
